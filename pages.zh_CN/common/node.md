# node

> 服务端的 JavaScript 框架  （Nodejs）
> 项目主页： <https://nodejsorg> 

- 运行一个 JavaScript 脚本文件：

`node {{file}}.js`

- 启动 REPL（交互式 shell）：

`node`

- 评估测试一段代码：

`node -e "{{code}}"`

- 评估测试并输出结果，在查看 Node 的依赖版本时非常有用：

`node -p "{{process.versions}}"`
