# cabal

> Haskell 包基础设施（ cabal ）的命令行接口
> 从 Hackage 软件包存储库管理 haskell 项目和 cabal 软件包

- 搜索并列出 Hackage 包：

`cabal list {{search_string}}`

- 显示有关包的信息：

`cabal info {{package_name}}`

- 下载并安装软件包：

`cabal install {{package_name}}`

- 在当前目录中新建 Haskell 项目：

`cabal init`

- 在当前目录中生成项目：

`cabal build`

- 在当前目录中运行项目的测试：

`cabal test`
