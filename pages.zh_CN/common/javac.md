# javac

> Java 应用编译器

- 编译一个“.java”文件：

`javac {{file.java}}`

- 编译几个“.java”文件：

`javac {{file1.java}} {{file2.java}} {{file3.java}}`

- 编译在当前目录中的所有“.java”文件：

`javac {{*.java}}`

- 编译一个“.java”文件并将结果“class”文件输出到指定目录：

`javac -d {{path/to/some/directory}} {{file.java}}`
