# php artisan

> Laravel 的 Artisan 命令行接口

- 为当前的 Laravel 应用启动 PHP 内置的 web 服务器：

`php artisan serve`

- 启动交互式 PHP 命令行接口：

`php artisan tinker`

- 生成新的 Eloquent 模型类及migration、factory 和 resource 控制器：

`php artisan make:model {{ModelName}} --all`

- 显示所有可用的命令列表：

`php artisan help`
