# python

> Python 语言解释器

- 调用 Python 交互式解释器（REPL）：

`python`

- 执行指定 Python 脚本文件：

`python {{script.py}}`

- 将脚本作为交互式 shell 的一部分执行：

`python -i {{script.py}}`

- 执行 Python 表达式：

`python -c "{{expression}}"`

- 以脚本方式运行库模块（结束选项列表，其后跟的是模块的参数）：

`python -m {{module}} {{arguments}}`

- 以交互方式调试 Python 脚本：

`python -m pdb {{script.py}}`
