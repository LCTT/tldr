# nimble

> Nim 编程语言的包管理工具
> 管理 Nim 工程和他们的依赖关系

- 查找包：

`nimble search {{search_string}}`

- 安装包：

`nimble install {{package_name}}`

- 列出已安装的包：

`nimble list -i`

- 在当前目录新建一个 Nimble 包：

`nimble init`

- 构建一个 Nimble 包：

`nimble build`

- 安装一个 Nimble 包：

`nimble install`
