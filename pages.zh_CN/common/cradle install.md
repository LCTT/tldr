# cradle install

> 安装 Cradle PHP 框架组件

- 安装 Cradle 的组件（系统将提示用户提供更多详细信息）：

`cradle install`

- 强制覆盖文件：

`cradle install --force`

- 跳过运行 SQL 迁移：

`cradle install --skip-sql`

- 跳过运行包更新：

`cradle install --skip-versioning`

- 使用特定的数据库细节：

`cradle install -h {{hostname}} -u {{username}} -p {{password}}`
