# code

> Visual Studio Code 编辑器
> 主页： <https://githubcom/microsoft/vscode>

- 打开 VS Code：

`code`

- 在 VS Code 中打开当前目录：

`code .`

- 在 VS Code 中打开指定文件或目录：

`code {{path/to/file_or_directory}}`

- 在当前 VS Code 窗口打开一个文件或目录：

`code --reuse-window {{path/to/file_or_directory}}`

- 在 VS Code 中比较两个文件：

`code -d {{file1}} {{file2}}`
