# dotnet

> 交叉平台 NET 命令行工具用于 NET Core 

- 初始化一个新的 .NET 项目：

`dotnet new {{template_short_name}}`

- 还原 nuget 包：

`dotnet restore`

- 在当前目录中编译并执行 .NET 项目：

`dotnet run`

- 运行一个打包好的 .NET 程序（只需要运行时环境，其它的命令需要安装 .NET 核心 SDK）：

`dotnet {{path/to/application.dll}}`
