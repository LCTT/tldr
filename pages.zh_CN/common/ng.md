# ng

> 创建和管理 Angular 应用的命令行界面（CLI）

- 在当前文件夹创建 Angular 应用：

`ng new {{project_name}}`

- 为应用添加组件：

`ng generate component {{component_name}}`

- 为应用添加类：

`ng generate class {{class_name}}`

- 为应用添加新指令：

`ng generate directive {{directive_name}}`

- 在根目录用接下来的命令运行应用：

`ng serve`

- 构建应用：

`ng build`

- 运行单元测试：

`ng test`

- 显示安装的 Angular 版本：

`ng version`
