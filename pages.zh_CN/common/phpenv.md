# phpenv

> 用于 PHP 开发的版本管理器

- 全局模式安装指定版本的PHP：

`phpenv install {{version}}`

- 刷新 `phpenv` 已知的所有 PHP 二进制文件的 shim 文件：

`phpenv rehash`

- 列出所有已安装的 PHP 版本：

`phpenv versions`

- 显示当前使用的 PHP 版本：

`phpenv version`

- 设置全局 PHP 版本：

`phpenv global {{version}}`

- 为本地指定一个替代全局的 PHP 版本：

`phpenv local {{version}}`

- 取消指定本地 PHP 版本：

`phpenv local --unset`
