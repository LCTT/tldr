# drush

> Drupal 的命令行 shell 和脚本接口

- 下载模块 foo：

`drush dl {{foo}}`

- 下载模块 foo 的 7.x-2.1-beta1 版本：

`drush dl {{foo}}-7.x-2.1-beta1`

- 启用模块 foo：

`drush en {{foo}}`

- 禁用模块 foo：

`drush dis {{foo}}`

- 清除所有缓存：

`drush cc all`

- 清除 CSS 和 JavaScript 缓存：

`drush cc css-js`
