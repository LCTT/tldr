# git remote

> 管理追踪远程仓库
> 主页地址: <https://git-scmcom/docs/git-remote>

- 列出远程仓库的名字和 URL：

`git remote -v`

- 添加远程仓库：

`git remote add {{remote_name}} {{remote_url}}`

- 修改远程仓库的 URL：

`git remote set-url {{remote_name}} {{new_url}}`

- 移除指定的远程仓库：

`git remote remove {{remote_name}}`

- 修改远程仓库名称：

`git remote rename {{old_name}} {{new_name}}`
