# ember

> Emberjs 的命令行工具
> 用来创建和维护 Emberjs 应用

- 创建一个新的 Ember 应用：

`ember new {{my_new_app}}`

- 创建一个新的 Ember 扩展：

`ember addon {{my_new_addon}}`

- 给项目打包：

`ember build`

- 以生产模式打包项目：

`ember build -prod`

- 运行开发服务器：

`ember serve`

- 运行测试套件：

`ember test`

- 使用预置模型生成一些演示路由或组件：

`ember generate {{type}} {{name}}`

- 安装 ember-cli 扩展：

`ember install {{name_of_addon}}`
