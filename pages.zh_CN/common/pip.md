# pip

> Python 包管理器
> 主页： <https://pippypaio>

- 安装一个包：

`pip install {{package_name}}`

- 安装指定版本的包：

`pip install {{package_name}}=={{package_version}}`

- 升级某个包：

`pip install -U {{package_name}}`

- 卸载某个包：

`pip uninstall {{package_name}}`

- 将已安装的包列表保存到文件中：

`pip freeze > {{requirements.txt}}`

- 从文件读取依赖包并安装：

`pip install -r {{requirements.txt}}`
