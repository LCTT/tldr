# grunt

> JavaScript 的一个自动化任务构建工具

- 执行默认任务流程：

`grunt`

- 执行一个或多个任务（多个任务名由空格隔开）：

`grunt {{task_name}}`

- 使用指定的配置文件代替默认配置文件执行任务：

`grunt --gruntfile {{path/to/file}}`

- 为相对路径文件名设置基础路径：

`grunt --base {{path/to/directory}}`

- 在指定目录里扫描任务：

`grunt --tasks {{path/to/directory}}`

- 执行干操作，不写入文件：

`grunt --no-write`

- 列出所有可用选项：

`grunt --help`
