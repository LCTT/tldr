# mocha

> 执行 Mocha JavaScript 测试运行器

- 使用默认配置或在 mocha.opts 中配置运行测试：

`mocha`

- 运行包含在特定位置的测试：

`mocha {{directory/with/tests}}`

- 运行与特定 grep 模式匹配的测试：

`mocha --grep {{^regex$}}`

- 对当前目录和最初的 JavaScript 文件的更改运行测试：

`mocha --watch`

- 与特定 reporter 一起运行测试：

`mocha --reporter {{reporter}}`
