# cradle deploy

> 管理 Cradle 部署

- 将 Cradle 部署到服务器：

`cradle deploy production`

- 将静态资源部署到 Amazon S3：

`cradle deploy s3`

- 部署静态资产，包括 Yarn 的 components 目录：

`cradle deploy s3 --include-yarn`

- 部署静态资产，包括 upload 目录：

`cradle deploy s3 --include-upload`
