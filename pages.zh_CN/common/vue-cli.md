# vue-cli

> Vuejs 项目脚手架的简易命令行界面
> 官方模版包括: webpack、webpack-simple,、browserify、browserify-simple、simple

- 创建一个新的 vue 项目：

`vue init {{template}} {{project_name}}`

- 使用本地模版创建一个新的 vue 项目：

`vue init {{path/to/template_folder}} {{project_name}}`

- 使用 GitHub 上的模版来创建项目：

`vue init {{username}}/{{repo}} {{project_name}}`
