# meteor

> 用于构建 Web 应用程序的全栈 javascript 平台

- 在开发模式下从其根目录运行 meteor 项目：

`meteor`

- 在给定目录下创建项目：

`meteor create {{path/to/directory}}`

- 显示项目当前使用的包列表：

`meteor list`

- 将包添加到项目中：

`meteor add {{package_name}}`

- 从项目中删除包：

`meteor remove {{package_name}}`

- 在给定目录下创建项目的生产版本作为 tarball ：

`meteor build {{path/to/directory}}`
