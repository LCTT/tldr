# http-prompt

> 一个交互式的命令行 HTTP 客户端，包含自动补全以及语法高亮

- 启动一个会话过程，其省却的 URL 在 http://localhost:8000 或是上一个会话过程：

`http-prompt`

- 为一个指定 url 链接开启会话：

`http-prompt {{http://example.com}}`

- 启动一个会话并设置一些初始选项：

`http-prompt {{localhost:8000/api}} --auth {{username:password}}`
