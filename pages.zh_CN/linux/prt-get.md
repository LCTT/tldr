# prt-get

> 高级CRUX包管理器

- 安装某个包：

`prt-get install {{package_name}}`

- 安装某个包并处理依赖：

`prt-get depinst {{package_name}}`

- 手动更新某个包：

`prt-get upgrade {{package_name}}`

- 删除某个包：

`prt-get remove {{package_name}}`

- 从本地端口树更新系统：

`prt-get sysup`

- 搜索端口数：

`prt-get search {{package_name}}`

- 在包内查找文件：

`prt-get fsearch {{file}}`
