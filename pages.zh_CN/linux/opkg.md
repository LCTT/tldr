# opkg

> 用于安装 OpenWrt 包的轻量级包管理器

- 安装程序包：

`opkg install {{package}}`

- 删除一个程序包：

`opkg remove {{package}}`

- 更新可用包的列表：

`opkg update`

- 升级所有已安装的包：

`opkg upgrade`

- 升级一个或多个特定包：

`opkg upgrade {{package(s)}}`

- 显示特定包的信息：

`opkg info {{package}}`

- 列出所有可用的包：

`opkg list`
